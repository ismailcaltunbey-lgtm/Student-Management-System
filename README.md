# Student Management System

A simple console-based application developed with C# to manage student records. This project focuses on basic CRUD operations and object-oriented programming (OOP) principles.

## Features
- **Add Student:** Register new students by entering their ID, Name, and Department.
- **List Students:** View all registered students in the system.
- **Interactive Menu:** Easy-to-use console menu for navigation.

## Technologies Used
- C#
- .NET Framework

## How to Run
1. Clone this repository to your local machine.
2. Open `Program.cs` in Visual Studio or any C# compatible editor.
3. Build and run the project.
